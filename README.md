## **Project Idea**

The idea of this project is to test the demo e-commerce website Automation
Exercise.
Automation Exercise provides a wide range of real-world e-commerce features
such as user authentication, product browsing, cart management, checkout, and
account handling. These make it an ideal platform for practicing manual testing,
test automation, and API validation.

## **Roles & Responsibilities**

Each team member will perform **Manual, Automation, and API Testing** for their assigned features.

### **1. Mahfouz Sherif**

**Feature:** User Authentication

* **Manual:** Test registration, login, and logout workflows.
* **Automation:** Automate login/logout with valid & invalid data.
* **API:** Validate `/api/createAccount` and `/api/login` endpoints.

---

### **2. Ziad Badr**

**Feature:** Product Browsing & Search

* **Manual:** Test product listing, categories, and search bar.
* **Automation:** Automate product search and filter workflows.
